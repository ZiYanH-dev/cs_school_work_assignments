# Programming Assignments

This repository stores school coding assignments organized by course number.

## Course folders

- `1023/` - Python programming, including games, GUI, OOP, and practice with common libraries
- `2011/` - C++ coursework focused on fundamentals, labs, and algorithmic/problem-solving assignments
- `2211/` - AI/ML-oriented coursework with data analysis and model-building tasks

## Notes

- Each course has a course-level `README.md` for a general overview.
- Assignment folders may grow over time, so this documentation stays general instead of hardcoding lab/PA counts.
- No per-lab or per-PA README files are required unless you choose to add them later.
