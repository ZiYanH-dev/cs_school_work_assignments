# Course 2011

## Overview

This folder contains Course 2011 assignments and related resources.

The work here is mainly C++-based. From the current folders/files, this course includes:

- foundational labs on core C++ topics
- programming assignments (`pa`) with structured problem-solving tasks
- recursive pathfinding/maze-style assignment work (for example, in `pa2`)
- starter/skeleton materials and local test scripts for assignment checking
